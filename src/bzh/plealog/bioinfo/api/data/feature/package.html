This package contains the components used to model Feature Data. 

Following the NCBI/EBI/DDBJ feature data model, this package provides the following 
components:

- FeatureTable: contains a set of features that are mapped on a sequence. A Feature
  Table contains a list of Feature objects.
- Feature: a single feature. Contains a location and a set of Qualifiers.
- FeatureLocation, FRange and FPosition: they are used only when a feature location
  cannot be represented with a single pair of from and to values.
- Qualifier: a key/value pair of strings.
