This package contains implements a SearchResult loader capable of handling NCBI XML Blast Results.

When using BLAST+, simply use argument "-outfmt 5" to create legacy XML result file readable by this loader. As of July 2016, this option is still available in BLAST+ 2.4.0.

It is worth noting that this data model only handles the legacy NCBI XML format, i.e. NCBI_BlastOutput.dtd, and NOT the new XML2 format introduced with BLAST+ 2.2.31 on June 2015.

All classes in this package are generated from NCBI XML DTD using the Castor Framework.
